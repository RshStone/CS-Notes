**词法分析**

从左到右逐行扫描程序字符： 识别出各个单词，确定单词类型，之后将识别出单词转为统一机内表示-词法单元(token)形式

**一个思考：如何转成token的呢？**

token: <种别码，属性值>

**语法分析**

将token序列中识别各类短语，构造语法分析树



文法是由一系列规则构成的

<D> => <T> <IDS>;

<T> => int | real | char | bool

<IDS> => id | <IDS>,id
